# ch-ai-tanya

*Chaitanya* (Sanskrit: consciousness-force), with AI sitting inside the
word. A research vault on model psychology — the study of
psychological-level phenomena in large language models.

## Status

v0. Under active construction. See `schema.md` for structure and
`CLAUDE.md` for conventions.

## What's here

Findings, concepts, and perspectives on character, persona, emotion,
introspection, motivation, deception, and the internal structures that
give rise to them in modern LLMs. AI-curated, human-edited, public as a
side effect of doing the work in the open.
